============
Installation
============

Requires:

* python 3.7 or greater

Install from pip
================

* (recommended) Create a virtual environment (e.g., Mamba)
* Install usinng pip::

    pip install nemi-learn

Install from source
===================
* Clone the repository::

    git clone https://github.com/maikejulie/NEMI.git

* (recommended) Create a virtual environment (e.g., Mamba)
* Install using pip::

    pip install -e .
